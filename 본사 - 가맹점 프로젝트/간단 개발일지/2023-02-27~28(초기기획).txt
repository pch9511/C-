Web 
-- 가맹점 -- 
-> 가맹점등록  = 가맹점명, 가맹점주 이름, 가맹점주 연락처,사업자등록번호,대표연락처,주소(시,동) -> 본사(administrator 에서 승인)
-> 로그인 ID,PW(본사에서 승인났을때 부여) 가맹점주 이름,연락처는 임시로그인(승인이전)

승인 이전 -> 기능을 제한
승인 이후 -> 기능 확장 개념 


-> 로그인 이후 기능 
-> 가맹점 정보 조회,수정, 탈퇴(승인시 탈퇴) 
-> 발주서관리(주문(전,후),배송(전,후),결제,반품)

WinForm 
-- 본사 -- 

1. 가맹점 관리 ( 등록,수정,삭제,조회)

2. 재고관리   
   -> 1. 카테고리 등,수,삭,조
   -> 2. 입/출고 관리 
   -> 3. 거래처(매입관련) -> 사업자,품목,입출고,발주,결제 관리 시스템


3. 가맹점 업무
   -> 1. 발주, 주문 등

4. 재무상태 - 가맹점별 매출 조회, 총 매출, 총 이익 등 

5. 가맹점 채팅창 (멀티채팅) -> WinForm

재고관리, 채팅창 -> 윈폼
